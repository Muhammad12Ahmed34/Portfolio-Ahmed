# Muhammad Ahmed — Portfolio Website
 
A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Clean dark theme, fully responsive, no frameworks.
 
---
 
## 🚀 Live Preview
 
> (https://muhammad12ahmed34.github.io/Portfolio-Ahmed/)
 
---
 
## 📁 Project Structure
 
```
portfolio/
├── index.html        # Main HTML file
├── style.css         # All styling
├── script.js         # Mobile menu + typewriter effect
└── assets/
    ├── My Image.png        # Profile photo
    ├── Ahmed Tech logo.png # Favicon
    └── CV.pdf              # Downloadable resume
```
 
---
 
## ✨ Features
 
- **Typewriter Effect** — Rotating identity text in the hero section
- **Responsive Design** — Works on mobile, tablet, and desktop
- **Mobile Menu** — Slide-in nav with open/close toggle
- **Scroll Shadow** — Header shadow activates on scroll
- **Smooth Scrolling** — Anchor links scroll smoothly
- **Dark Theme** — Full black/card dark UI with green accent
---
 
## 🛠️ Built With
 
- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) — Google Fonts

## ⚙️ Setup
 
No installation needed. Just clone and open in browser.
 
```bash
git clone https://github.com/Muhammad12Ahmed34/portfolio.git
cd portfolio
# Open index.html in your browser
```
 
Or use VS Code with **Live Server** extension for live reload during development.
